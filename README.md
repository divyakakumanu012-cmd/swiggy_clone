# 🍔 Swiggy Clone

A simple full-stack Swiggy Clone project built using **React.js**, **Node.js**, and **Express.js**. This project displays food items from a backend API and shows them in a responsive frontend UI.

---

# 🚀 Features

* React frontend
* Node.js + Express backend
* REST API integration
* Food item cards
* Responsive UI
* Add to Cart button UI
* Clean project structure
* GitHub integration

---

# 🛠️ Tech Stack

## Frontend

* React.js
* HTML5
* CSS3
* JavaScript

## Backend

* Node.js
* Express.js
* CORS

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/divyakakumanu012-cmd/swiggy_clone.git
```

```bash
cd swiggy_clone
```

---

# 🔧 Backend Setup

```bash
cd backend
```

Install dependencies:

```bash
npm install
```

Run backend server:

```bash
node server.js
```

Backend runs on:

```bash
http://localhost:5000
```

API Endpoint:

```bash
http://localhost:5000/food-items
```

---

# 💻 Frontend Setup

Open a new terminal:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Run frontend:

```bash
npm start
```

Frontend runs on:

```bash
http://localhost:3000
```
